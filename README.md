The Iris dataset is a classic in machine learning, introduced by Ronald Fisher in 1936. This project demonstrates how a Decision Tree can classify iris species based on four features: sepal length, sepal width, petal length, and petal width. Decision trees are highly interpretable â€“ you can trace the path from root to leaf to understand why a particular sample is classified as setosa, versicolor, or virginica.
